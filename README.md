# Kick4Kause

Summer soccer camp website built with [Next.js](https://nextjs.org).

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Deploy on [Vercel](https://vercel.com/new) by importing this repository. Set environment variables `AUTH_SECRET` and `ADMIN_PASSWORD` if you use the staff admin area.
